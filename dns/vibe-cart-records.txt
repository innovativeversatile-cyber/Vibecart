# In Netlify: Site settings → Domain management → Add custom domain → vibe-cart.com

# Railway will show a target hostname, often like:  xxxxx.up.railway.app
#
#   Type: CNAME    Name: api    Target: <copy from Railway dashboard>
#
# --- 3) Email later (optional) ---
