    Security overview

    The same protections apply sitewide — we only surface the detail here so the storefront stays lightweight.

    • Owner-only admin with multi-factor authentication and hardware key support.
    • Country-aware fraud detection and transaction monitoring.
    • Buyer and seller protection with suspicious-activity checks, identity verification, and high-risk order review.
    • Escalation and dispute workflow with temporary account lock for suspected fraud.
    • Payments are routed only through vetted providers with encryption, 3D Secure, anti-fraud scoring, and tokenized transactions.
    • Delivery uses trusted logistics partners with tracking, delivery verification, and route risk controls.
    • Encrypted traffic (HTTPS), secure cookies, and strict security headers.
    • Role-based permissions and immutable audit logs.
    • Legal-only marketplace with moderation, reporting controls, and proactive listing removal for unlawful content.
